# Audio Visualizer

本プロジェクトは、リアルタイムでオーディオ入力を解析・視覚化する**オーディオビジュアライザー**です。  
Windows環境での実行を前提としています。

---

## 📦 導入手順（ZIP形式でダウンロードする場合）

1. このリポジトリ右上の「Code」ボタンをクリックし、「Download ZIP」を選択してZIPファイルをダウンロードします。
2. ダウンロードしたZIPを右クリックして「すべて展開」し、任意の場所に解凍してください。

---

## ⚡ デフォルトの EXE を使う場合（カスタマイズ不要）

デフォルトの設定のままでよい人は、以下の手順で EXE を起動するだけでビジュアライザーを使用できます。

### 1. 音声デバイスの準備

#### 有線イヤホン／スピーカー／ヘッドホンを使用する場合

1. 「サウンド設定」>「録音」タブを開く
2. 「ステレオミキサー」を右クリックし「有効化」
3. 「既定のデバイスに設定」は不要

> 💡 ノートパソコンではステレオミキサーが無くても動作する場合があります。

#### Bluetooth機器や VB-CABLE を使用する場合

1. [VB-CABLE公式サイト](https://vb-audio.com/Cable/) から VB-CABLE をダウンロード
2. ZIP を解凍し `VBCABLE_Setup.exe` または `VBCABLE_Setup_x64.exe` を **右クリック→「管理者として実行」** してインストール
3. 「サウンド設定」で「CABLE Input」「CABLE Output」が追加されていることを確認
4. 使用するアプリ（Spotify など）の出力先を **「CABLE Input」** に設定
5. ビジュアライザーの入力には **「CABLE Output」** を選択し、出力にスピーカーを設定

### 2. EXE の起動

1. ZIP 展開フォルダ内の `Visualizer.exe` をダブルクリック
2. ビジュアライザーが起動し、音に合わせてリアルタイムで反応します

> 💡 タスクバーのアイコンからもウィンドウを閉じられます

### 3. EXE を自作したい場合（ビルド手順）

1. Python 環境で [PyInstaller](https://pyinstaller.org/) をインストール
2. プロジェクトのルートフォルダで以下を実行：

```powershell
pyinstaller --onefile --windowed --name AudioVisualizer auto_mv.py
````

3. `dist` フォルダ内に `main.exe`（または指定した名前）が生成されます
4. この EXE を ZIP に同梱して配布すれば、Python 環境がなくても動作します

## 🛠 カスタマイズする人向け（VSCode + 仮想環境）

> 💡 ここから下は、設定を変更したり開発したい人向けの手順です

### 1. VSCodeでプロジェクトを開く

* 解凍したプロジェクトフォルダを VSCode で開きます
* ターミナルを PowerShell に設定（「端末」>「新しいターミナル」> プロファイル選択）

### 2. 仮想環境の作成と有効化

```powershell
# プロジェクト直下に仮想環境を作成
python -m venv muvenv

# 仮想環境を有効化
.\muvenv\Scripts\activate
```

> 💡 仮想環境を切り替える場合は `deactivate` で無効化

### 3. 依存ライブラリのインストール（uv sync方式）

```powershell
# 仮想環境がアクティブな状態で
uv sync --active
```

### 4. Python スクリプトの実行

1. VSCode 右上の「▶ 実行」ボタンをクリック
2. 入力デバイスと出力デバイスを選択
3. デバイスが確定したら 3 秒ほど待ち、ウィンドウをクリックやキー入力
4. ビジュアライザーが動作開始

## 🛠 トラブルシューティング

| 問題                   | 解決方法                                        |
| -------------------- | ------------------------------------------- |
| 「デバイスが見つかりません」と表示される | 入出力デバイスが正しく接続されているか確認し、再起動してください            |
| ステレオミキサーが表示されない      | 「無効なデバイスを表示する」にチェックを入れて確認してください             |
| 音が出ない／反応しない          | 入出力の設定が正しいか、VB-CABLEなどのルーティングが正しいかを確認してください |
